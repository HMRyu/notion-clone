## Notion Clone Coding

실시간 협업 문서 관리를 할 수 있는 Notion 서비스를 클론 코딩해보며 이해도를 높이려 하였습니다.

## Tech Stack

- Next.js
- TypeScript
- Liveblocks.io
- Lexical Editor
- Tailwind CSS
- shadcn

## Quick Start

### Clone
```
git clone https://github.com/HMRyu/notion-clone.git
```

### Installation
```
npm install
```

### Run
```
npm run dev
```
